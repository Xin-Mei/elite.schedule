院區設定

員工名單與分組
分組決定休假日歸屬 (A/B/C/D)。同一組的員工互為輪班搭檔。
💡 區間設定 (人員替換):每位員工可指定生效期間。同一組若有交接,新增多筆即可。
  範例:櫃台 A 組 → 小玫 (5/10~5/31 離職) + 小貝 (6/1~7/4 接班) → 系統會依日期自動套用對應員工。
  留空 = 該方向無限制 (空 from = 不限早期 / 空 to = 一直生效)。
排班規則 — 每日各班人數
若該職位無午班,將午班人數設為 0 即可。
每日出勤人數:若該職位不分早晚 (例如:藥師整天班),設為 1,班表只顯示「出勤」一格。
輪班循環設定 (Pattern)
定義一個固定循環,系統會依此自動推出每位員工每天該上的班別。
每格選 早 / 午 / 晚 / 休,例如 早 早 晚 晚 休 休 = 6 天循環。
💡 若想統一管理同職位的循環,可在 Google Sheet 維護:
  • _輪班循環 — pattern (早早晚晚休休)
  • _員工起始 — 各組 ABC 的起始位置 (新院區一鍵套用)
員工起始位置 (依此期八週)
新一期 8 週開始的第 1 天,每位員工從循環的第幾格起算。
例:循環是「早早晚晚休休」,小明設為 3,則第 1 天他上晚班、第 2 天晚班、第 3 天休假...
ℹ️ 工讀 為兼職人員,上班時間不固定。下方設定週一 ~ 週日的標準上班時段,系統會以此為預設;若某週有特殊情況可在排班表中個別調整。
時段模式
選擇此院區工讀的時段制:三時段 (早/午/晚) 或 兩時段 (早/晚)
每週上班時間維護 (週一 ~ 週日)
勾選有上班的時段並填寫該時段需要的人數;未勾選 = 該日無班。
這是「標準週模板」,實際排班時可在月曆中個別覆蓋。
星期 早班 午班 晚班 說明
八週變形工時區間
最多保留 3 個區間 (歷史 / 當前 / 未來) 供查閱。
💡 若想統一管理所有院區的區間,可在 Google Sheet 的 _工時區間 工作簿維護,系統會自動套用作為新院區的預設值。
